### Installation & Setup

  Clone this repository and install its dependencies.

        > git clone https://github.com/temmietope/quiz-app-with-reactjs

        > cd quiz-app-with-reactjs

        > npm install

        > npm start

  Application is live on port 3000

        http://localhost:3000
